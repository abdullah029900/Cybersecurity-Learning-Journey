# IP Address

An IP address is a unique number given to a device on a network.

## Example:
192.168.1.1

## Key Points:
- Has 4 parts (octets)
- Identifies a device on a network
- Can change when network changes

## Types:
- Public IP → Used on Internet
- Private IP → Used in local network (home/office)

## Example:
Your mobile and laptop get different IPs on same WiFi.
