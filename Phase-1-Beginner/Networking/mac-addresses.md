# MAC Address

A MAC address is a unique hardware address of a device.

## Example:
a4:c3:f0:85:ac:2d

## Key Points:
- Permanent (does not change easily)
- Assigned by manufacturer
- Used in local network communication

## Difference from IP:
- IP = changes with network
- MAC = fixed hardware identity

## Use case:
Routers use MAC address to identify devices inside a network
